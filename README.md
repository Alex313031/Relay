## download

[download the macOS `.app` here](https://github.com/brandly/irc/releases)!

for other platforms, you'll have to build your own.

## dev

```shell
$ npm install
$ npm start
```

to start this thing with react/redux devtools

```shell
$ npm run dev
```

## dist

to build an executable for your OS

```shell
$ npm run build
```
